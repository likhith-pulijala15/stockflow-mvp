# StockFlow MVP

Inventory Management SaaS MVP built as part of the Wexa AI Software Developer Intern technical assessment.

## Tech Stack

- React
- Tailwind CSS
- Node.js
- Express.js
- MySQL
- Sequelize
- JWT Authentication

## Features

- User Authentication
- Organization-based Multi-tenancy
- Product Management
- Inventory Tracking
- Dashboard Overview
- Low Stock Monitoring
