# API Guide

This document contains the API endpoints for the StockFlow MVP application.

# Authentication

## Signup

POST /api/auth/signup

## Login

POST /api/auth/login

# Products

## Get Products

GET /api/products

## Create Product

POST /api/products

## Update Product

PUT /api/products/:id

## Delete Product

DELETE /api/products/:id

# Dashboard

## Get Dashboard Summary

GET /api/dashboard
